# Django Server Setup Automation

Цей проект надає інструкції та скрипт для автоматизованого налаштування Django сервера. Він допоможе встановити всі необхідні залежності, налаштувати базу даних і запустити сервер, мінімізуючи ручні налаштування.

## Як користуватися

1. **Запустіть налаштувальний скрипт**:
   - У Windows відкрийте командний рядок.
   - Перейдіть до директорії проекту та запустіть скрипт:
     ```bash
     python setup_script.py
     ```
   - Цей скрипт:
     - Створить віртуальне середовище.
     - Встановить всі бібліотеки з `requirements.txt`.
     - Налаштує файл `.env` з параметром `DEBUG=True`.

2. **Налаштування бази даних**: Скрипт запитає, чи хочете ви залишити тестову базу або створити нову:
   - **Залишити базу**: обирайте цей варіант, щоб залишити існуючі дані.
   - **Очистити базу**: скрипт видалить файл `db.sqlite3`, виконає `makemigrations` та `migrate` для створення нової бази.

3. **Створення суперкористувача** (опціонально): Вам буде запропоновано створити суперкористувача для доступу до адміністративної панелі Django.

4. **Запуск сервера**: Після завершення налаштувань скористайтеся окремим скриптом для запуску сервера:
   ```bash
   python start_server.py
